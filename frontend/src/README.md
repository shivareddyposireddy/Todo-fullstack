# Full Stack Todo Application

A full-stack Todo application built using:

- Spring Boot (Backend REST API)
- React + Vite (Frontend)
- H2 Database

---

## Features

- Create Todo
- View Todos
- Update Todo
- Delete Todo
- Mark as Completed

---

## Tech Stack

### Backend
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

### Frontend
- React (Hooks)
- Axios
- Vite

---

## Project Structure

todo-fullstack/
├── backend/   (Spring Boot API)
└── frontend/  (React App)

---

## Run Backend

cd backend  
mvn spring-boot:run  

Runs at: http://localhost:8080

---

## Run Frontend

cd frontend  
npm install  
npm run dev  

Runs at: http://localhost:5173
